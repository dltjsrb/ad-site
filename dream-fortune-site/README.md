# 꿈해몽 & 운세 정보 사이트 - 제작 현황

## 실무 배포 메모

- 콘텐츠 검색/아카이브 데이터는 `assets/js/site-data.js`를 기준으로 관리합니다.
- 배포 전에는 `powershell -ExecutionPolicy Bypass -File .\scripts\generate-seo-assets.ps1 -SiteUrl "https://실제도메인"`을 실행해 `sitemap.xml`, `robots.txt`, `site.webmanifest`, 각 페이지의 canonical/OG/twitter 메타를 갱신합니다.
- 현재 기본 도메인은 예시용 `https://dreamfortunearchive.example`로 들어가 있으니, 실제 배포 도메인으로 바꿔서 생성해야 합니다.

## 📝 작성 원칙 (기획서 준수)

### ✅ 해야 할 일
- 문장을 쉽고 명확하게
- 각 항목마다 충분한 설명 (최소 100자)
- 개인의 상황을 존중하는 표현
- 정보 전달에 집중

### ❌ 하면 안 될 일
- "100% 적중", "반드시" 같은 표현
- 의료/법률 조언처럼 보이는 단정
- 불안감 조장, 공포 표현
- 과도한 신비로움
- 반복되는 패턴의 자동 생성 느낌
